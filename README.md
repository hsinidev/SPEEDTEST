# Doodax SPEEDTEST

A modern, single-page web application for measuring internet connection speed, including download, upload, latency, and jitter. This project is built with vanilla JavaScript, Tailwind CSS, and leverages the powerful `@cloudflare/speedtest` module.

## Live Demo

You can try out the live version of the speed test here: [speedtester.doodax.com](https://speedtester.doodax.com)

## Features

-   **Comprehensive Metrics:** Measures download speed, upload speed, latency (ping), and jitter.
-   **Real-time Updates:** Displays speed test results as they are being measured.
-   **Server Information:** Shows details about the test server location and your network.
-   **Responsive Design:** A clean, mobile-friendly interface that works on all devices.
-   **Visually Engaging:** Features an animated galaxy background for a modern user experience.
-   **No Backend Required:** Runs entirely in the browser as a single HTML file.
-   **SEO Optimized:** Includes metadata, `robots.txt`, and `sitemap.xml` for better search engine visibility.

## Tech Stack

-   **HTML5**
-   **CSS3** with **Tailwind CSS**
-   **JavaScript (ESM)**
-   **[@cloudflare/speedtest](https://github.com/cloudflare/speedtest)** module for the core testing logic.

## Getting Started

This project is designed for simplicity and requires no build steps or local server.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/your-repo-name.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd your-repo-name
    ```
3.  **Open the `index.html` file:**
    Simply open the `index.html` file in your favorite web browser (e.g., Chrome, Firefox, Safari).

That's it! The speed test is ready to use.
